Taiwan Basketball API Response Spec

Base URL: http://localhost:3100/api

所有一般 API 成功回傳格式為 { "success": true, "code": 200, "message": "請求成功", "data": ... }，錯誤格式為 { "success": false, "code": 400, "error": { "message": "..." } }。

共用資料型別 Health Teams Schedule Games Errors

共用資料型別

Team

欄位 型別 說明
id number 球隊資料庫 id
leagueId number 聯盟資料庫 id
name string 球隊全名，例如 桃園璞園領航猿
shortName string | null 短名，例如 領航猿
logoUrl string | null 前端 public 圖片路徑，例如 /%E6%A1%83...webp
league object { id, code, name }

Game

欄位 型別 說明
id number 比賽資料庫 id
externalGameId string | null PLG 官方 game id，例如 662
gameCode string | null 官方場次代碼，例如 G01 或 PLAYOFFS Game1
gameDate string YYYY-MM-DD，例如 2025-10-26
gameTime string | null HH:mm:ss，例如 17:00:00
venue string | null 比賽場地，例如 桃園市立綜合體育館
attendance number | null 進場人數；未公布或未開賽為 null
capacity number | null 場館容量；未公布或未開賽為 null
stage string preseason、regular_season、playoffs、finals
status string scheduled、live、final、postponed、cancelled
homeScore / awayScore number | null 尚未完賽時通常是 null
homeTeam / awayTeam object { id, name, shortName, logoUrl }

Health

GET /api/health

檢查 API 與 PostgreSQL 是否可用。

{
  "success": true,
  "code": 200,
  "message": "請求成功",
  "data": {
    "status": "ok",
    "timestamp": "2026-05-14T09:30:00.000Z"
  }
}

Teams

GET /api/teams

取得球隊列表。前端拿球隊 logo 主要用這支。

Query 型別 範例
leagueCode string PLG
leagueId number 1
q string 勇士
limit / offset number limit=50&offset=0

POST /api/teams

建立球隊，成功回 201 與 Team。

{
  "leagueId": 1,
  "name": "臺北富邦勇士",
  "shortName": "勇士",
  "logoUrl": "/%E5%AF%8C%E9%82%A6%E5%8B%87%E5%A3%AB.webp"
}

PATCH /api/teams/:id

更新球隊，成功回 Team。

{
  "shortName": "勇士",
  "logoUrl": "/taipei-braves.webp"
}

DELETE /api/teams/:id

刪除球隊。成功回 204 No Content，沒有 response body。

Schedule

前端賽程頁建議使用這支。回傳 Game array，主客隊都包含 logo。

stage string regular_season 或 playoffs
status string scheduled 或 final

Games

GET /api/games

底層 games 查詢，response 與 /api/schedule 相同。

支援 query：leagueCode、season、stage、status、teamId、from、to、limit、offset。

GET /api/games/:id

取得單一比賽。回傳 { "success": true, "code": 200, "message": "請求成功", "data": Game }。

GET /api/games/:id/boxscore

取得單一比賽 boxscore，包含兩隊總計、兩隊每節數據、球員全場數據與球員每節數據。

欄位 型別 說明
game Game 與 GET /api/games/:id 相同的比賽資料。
teams.away / teams.home object 主客隊資料；每隊都有 team、stats、periods、players。
teams.*.stats object | null 球隊全場總計。若尚未匯入 boxscore，會是 null。
teams.*.periods array 球隊每節數據，格式為 { period, stats }。
teams.*.players array 球員列表；每筆包含 player、全場 stats、每節 periods。

Errors

錯誤 response

Status 情境
400 validation failed、foreign key 不存在、資料不符合 constraint
404 找不到 route、找不到指定 team/game
409 unique constraint conflict
500 server/database unexpected error 
{
  "success": false,
  "code": 400,
  "error": {
    "message": "Validation failed",
    "details": [
      {
        "type": "field",
        "value": "bad-stage",
        "msg": "Invalid value",
        "path": "stage",
        "location": "query"
      }
    ]
  }
}
